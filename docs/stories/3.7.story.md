### Documentation Standards
ObjectScript class documentation should use DocBook format:
```objectscript
/// JSONSchema.Validator - Main entry point for JSON Schema validation
/// <p>
/// Provides stateless validation of JSON data against JSON Schema.
/// </p>
/// <b>Parameters:</b>
/// <ul>
/// <li>pJSON - JSON data to validate</li>
/// <li>pSchema - JSON Schema to validate against</li>
/// </ul>
```
